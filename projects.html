DHCP

The objective of this at-home lab was to configure Router-A as a DHCP server and Router-B as a DHCP relay agent. DHCP was set up for the 192.168.2.0/24 network, excluding 192.168.2.254 to reserve it as the default gateway. A point-to-point link between the routers used the 192.168.1.0/30 subnet, with .1 on Router-A and .2 on Router-B.

VLAN ROAS ACL

The objective of this at-home lab was to simulate a cost-effective workplace network using a Catalyst 2960 switch and a Cisco 2811 router. ACLs were configured to allow only the Management VLAN (192.168.192.0) full access, while others were restricted, and additional virtual PCs were added to mirror a real-world environment.
